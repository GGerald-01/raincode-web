## Estructura
```
raincode-web/
├── index.html
├── css/styles.css
├── js/main.js
├── js/three-scene.js
├── js/particles.js
├── assets/logo.svg
└── README.md
```

## Tecnologías
- HTML5 + CSS3 + Vanilla JS
- Three.js r128 (modelo 3D interactivo)
- Google Fonts (Inter)
- GitHub Pages (hosting)

## Desarrollo local
No requiere build. Abre `index.html` directamente en el navegador o sirve la carpeta:
```bash
python -m http.server 8000
```
